### Git(분산 버전 관리 프로그램)
* Git 역사 : unix 대형 컴퓨터 -> 가정용 -> 리눅스 토발즈 -> Linux
   -> GNU 재단(자유 소프트웨어(오픈소스) : 누구나 소스코드를 볼 수 있으며 무료로 사용(수정/사용/배포)가능한코드) -> Git

* 버전 : 컴퓨터 소프트웨어의 특정상태
* 관리 : 어떤 일의 사무, 시설이나 물건의 유지, 개량
* 프로그램 : 컴퓨터에서 실행될 때 특정 작업을 수행하는 일련의 명령어 집합
